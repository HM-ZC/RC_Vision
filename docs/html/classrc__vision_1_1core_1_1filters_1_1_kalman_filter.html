template<typename T, int StateDim, int MeasurementDim>
class rc_vision::core::filters::KalmanFilter< T, StateDim, MeasurementDim >

提供各种滤波器的实现，包括卡尔曼滤波器、扩展卡尔曼滤波器、无迹卡尔曼滤波器、 粒子滤波器和移动平均滤波器。

通用卡尔曼滤波器模板类。

模板参数
T数据类型（通常为 float 或 double）。
StateDim状态向量的维度。
MeasurementDim测量向量的维度。
