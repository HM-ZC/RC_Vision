\doxysubsection{详细描述}
图像处理类，封装了多种常用的图像操作和分析功能。 

该类提供了图像的加载、保存、转换、滤波、特征检测与描述、统计分析等多种功能， 以及图像的算术操作和模板匹配等高级操作。
