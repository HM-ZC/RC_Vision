\doxysubsection{详细描述}
配置管理类，支持 YAML 和 JSON 格式的配置文件加载与访问。 

该类提供了从文件中加载配置，并通过键值对的方式访问配置参数的功能。默认支持 YAML 格式，也可以通过设置 {\ttfamily format} 为 JSON 来加载 JSON 格式的配置文件。
